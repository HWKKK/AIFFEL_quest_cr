# AI 멘토링 앱

## 주요 기능
- 사용자가 관심 있는 분야를 선택하면, **AI가 해당 분야의 역사적 인물(멘토)**처럼 답변을 제공하는 대화형 학습 경험을 제공
- 멘토와의 학습에서 얻은 인사이트를 회고로 작성 및 대화 전문 저장

## 정보 구조도
  홈 화면 (HomeScreen)   
  ├── 멘토 리스트   
  │    ├── 관심 분야 선택   
  │    ├── AI 멘토 추천   
  │         ├── 멘토와 채팅   
  │              ├── 회고 작성 & 대화 저장   
  │              ├── 대화 리셋   
  │   
  ├── 내 멘토링  ← (하단 네비게이션 바)   
  │    ├── 저장된 멘토링 기록 리스트   
  │         ├── 특정 멘토링 상세보기   
  │              ├── 저장된 대화 내용 조회   
  │              ├── 회고 내용 확인
